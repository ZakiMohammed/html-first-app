Source Control/Version Control/scm (source control manager):
	- Control your code and projects (history, collaborations)
	- Git, SVN, Team Foundation, ...
	- Google Drive: Upload Folders (Bad)

GitHub
	- Online solution/portal for uploading your repository or projects
	- It uses Git in the background
	- GitHub Account
	- Git Commands:
		- Initial Commands:
			git init
			git remote add origin https://github.com/ZakiMohammed/html-first-app.git
		- Every time change commands:
			git add .
			git commit -m "this is my first commit"
			git push -u origin master
		- Cloning into another folder/machine:
			git clone https://github.com/ZakiMohammed/html-first-app.git
		- Pulling the changes:
			git pull
